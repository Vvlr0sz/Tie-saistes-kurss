# Tiešsaistes kursa cenu kalkulators

## Projekta apraksts
Šī mājaslapa ļauj lietotājam aprēķināt tiešsaistes kursa cenu atkarībā no izvēlētā līmeņa un dalībnieku skaita.

## Funkcionalitāte
- Lietotājs izvēlas kursa līmeni
- Ievada dalībnieku skaitu
- Sistēma aprēķina kopējo cenu
- Pie 5+ dalībniekiem tiek piemērota 10% atlaide
- Tiek veikta ievades datu validācija

## Lietošanas instrukcija
1. Atver index.html pārlūkā
2. Izvēlies kursa līmeni
3. Ievadi dalībnieku skaitu
4. Nospied pogu "Aprēķināt cenu"
5. Rezultāts parādīsies zem pogas
